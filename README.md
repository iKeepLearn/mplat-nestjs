## 功能介绍
此项目是[服务商微管家介绍](https://developers.weixin.qq.com/doc/oplatform/Third-party_Platforms/2.0/product/management-tools.html)的nestjs后端版本。不用改动前端。

## 使用注意
1. 本项目使用nestjs+prisma+postgresql。
  

2. 使用前准备工作

   - 修改 env 文件里数据库
   - 修改 prisma 目录下的 seed.ts 文件里的相关数据
   - 初始账号和密码都为 admin
